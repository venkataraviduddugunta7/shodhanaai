# POC Flow

## Demo Name

Shodhana AI - Duloxetine Trade Intelligence POC

## Business Pain

The sales team receives messy Excel files from export/import data sources. They manually clean product names, company names, units, prices, and duplicates before deciding which customers to approach.

## First Demo

Upload Excel -> clean products -> classify API vs pellets -> normalize importer/exporter -> convert quantity to KG -> calculate price/kg -> show dashboard -> rank opportunities -> generate customer-specific pitch.

## What To Say In The Demo

This is not replacing the sales team. It removes repeated manual Excel work and prepares sales-ready intelligence.

## What Data Shodhana Should Provide Next

- Product synonym master for Duloxetine API and pellets
- Company synonym master for known customers, competitors, and Shodhana entities
- Verified Shodhana product strengths, regulatory position, DMF/non-DMF rules, and capacity
- Current customer list
- Competitor list
- Approved pitch language
- Target price and margin guardrails

## Pilot Boundary

Build Grow first. Do not build Salesforce, Gmail, WhatsApp, or full RAG until the cleaning and opportunity engine is accepted.
