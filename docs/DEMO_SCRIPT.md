# Shodhana AI Demo Script

Use this in the client meeting.

## 1. Opening

Say:

```text
Today I am not showing a generic chatbot. I am showing the first working pilot of Shodhana AI for Duloxetine business development.
```

## 2. Explain The Problem

Say:

```text
The team spends too much time finding information, reading data, preparing emails, preparing presentations, and deciding which customer to focus on. The goal is to reduce that manual work and help the sales team focus on high-value conversion.
```

## 3. Explain The Pilot Scope

Say:

```text
We are not starting with every product. We are starting with Duloxetine API and Duloxetine Pellets. Once this works, the same system can be repeated for other products.
```

## 4. Show Knowledge Import

Action:

- Open the dashboard
- Click `Import Website`
- Click `Ingest Knowledge`

Say:

```text
The first layer of knowledge is Shodhana's public website. Later we will add internal approved documents, approvals, capacity, customer history, and product-specific notes.
```

## 5. Ask A Company Question

Question:

```text
Tell me about Shodhana company
```

Say:

```text
This shows the system can retrieve Shodhana-specific context instead of giving a generic answer.
```

## 6. Import Duloxetine Market Data

Action:

- Click `Import CSV`
- Show `Top Opportunities`

Say:

```text
Now the system has customer and market data for the Duloxetine pilot. It scores which customers should be approached first.
```

## 7. Explain Scoring

Say:

```text
The score is based on buyer type, estimated volume, trend, existing customer status, competitor presence, and suitability. This helps the sales team prioritize.
```

## 8. Generate A Pitch

Action:

- Click `Pitch` for a Tier 1 customer

Say:

```text
The system generates a customer-specific opportunity note, email draft, pitch direction, slide outline, and next action.
```

## 9. Be Honest About Current Stage

Say:

```text
This is a pilot. The quality of the output will become much stronger when we add Shodhana's internal approved product, regulatory, capacity, and market data.
```

## 10. Close With Roadmap

Say:

```text
The next step is to collect the internal Duloxetine data and real customer/export data. Then we will upgrade this from a working pilot into a production sales intelligence system with stronger AI generation, PPT automation, and CRM/email integration.
```
