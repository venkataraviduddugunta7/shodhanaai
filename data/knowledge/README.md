# Knowledge Folder

Put Shodhana knowledge files here as `.md`, `.txt`, or `.csv`.

Good first files:

- Company profile
- Facility and capacity notes
- Regulatory approvals
- Certificates
- Product portfolio
- Duloxetine API profile
- Duloxetine Pellets profile
- Competitor notes
- Existing pitch decks converted to text

Click **Ingest Knowledge** in the app after adding or editing files.

You can also click **Import Public Website** in the app. It will create:

```text
data/knowledge/shodhana_public_website.md
```

Use public website content as an official starting layer, then add verified internal facts for sales use.
